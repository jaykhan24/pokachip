# 🥔 포카칩 PokaChip

사진의 메타데이터(EXIF)를 자동으로 얹어 **인스타그램 스토리(1080×1920)** 이미지를 만드는 단일 HTML 도구입니다.
서버 없이 브라우저에서만 동작하므로 GitHub Pages에 올리면 바로 사용할 수 있습니다.

## 기능

- **중앙 카드 레이아웃** — 사진이 비율 카드(3:4 · 4:5 · 1:1 · 4:3 · 원본)로 항상 중앙 배치. 드래그로 구도(크롭) 조정, 핀치/휠로 확대
- **EXIF 자동 추출** — 촬영 날짜·요일·시간, GPS 위치(지역명 자동 변환), 촬영 기기, 조리개·셔터·ISO
- **자유 배치** — 모든 텍스트를 홀딩 앤 드래그로 이동, 더블탭(더블클릭)으로 직접 수정
- **배치 프리셋 & 기본값 저장** — 날짜+위치 한 줄·기기 한 줄을 사진 아래 / 위 / 위·아래 분할로 배치, 설정을 기본값으로 저장(localStorage)
- **텍스트 추가** — 한국어 폰트 6종(노토 산스, 나눔명조, 고운돋움, 나눔손글씨, 블랙한산스, 모노)
- **배경** — 메인 사진 블러 / 단색 컬러칩 / 그라데이션(사진에서 색 추출)
- **저장 & 공유** — JPG/PNG, 1x(1080×1920)/2x(2160×3840), 모바일에선 공유 시트로 인스타그램에 바로 전달
- **반응형** — 모바일 우선, PC에서는 중앙 미리보기 + 오른쪽 설정 패널

## 사용법

`index.html`을 브라우저로 열거나, GitHub Pages에 배포 후 접속하세요.

1. 화면을 탭해 사진 선택 (드래그 앤 드롭 가능)
2. 자동으로 표시된 날짜/위치/기기 정보를 드래그로 배치하거나 더블탭으로 수정
3. 하단 탭(사진·정보·텍스트·배경)에서 세부 조정
4. 우측 상단 **저장** 또는 **공유**

## GitHub Pages 배포

1. 이 폴더를 저장소에 푸시
2. Settings → Pages → Branch: `main` / root 선택
3. `https://<아이디>.github.io/<저장소명>/` 접속

## 참고

- **RAW/DNG 지원** — 아이폰 ProRAW, 갤럭시 Expert RAW(DNG) 및 TIFF 계열 RAW(NEF·CR2·ARW·RAF·RW2·ORF 등)는 파일 안에 담긴 가장 큰 JPEG 미리보기를 추출해 사용합니다. EXIF(날짜·위치·기기)도 RAW에서 그대로 읽습니다.
- 위치 지역명 변환은 [OpenStreetMap Nominatim](https://nominatim.org)을 사용합니다 (인터넷 연결 필요, 실패 시 좌표 표시)
- HEIC 파일은 브라우저가 지원하는 경우에만 열립니다 (아이폰 → JPG 변환 권장)
- 폰트는 Google Fonts CDN에서 불러옵니다
